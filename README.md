# Toxic Text Classifier RU

Модель для классификации русскоязычных сообщений 
на токсичные и нетоксичные.

## Используемая модель

ai-forever/ru-en-RoSBERTa

## Данные

Использовались датасеты:

- Mnwa/russian-toxic
- SetFit/amazon_massive_intent_ru-RU

## Результаты

Accuracy: 0.9849

Precision: 0.9873

Recall: 0.9825

F1-score: 0.9849

ROC-AUC: 0.9990

MCC: 0.9699

## Инференс

GPU: 17.3 мс  
CPU: 442 мс
